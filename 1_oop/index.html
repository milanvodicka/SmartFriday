Objekt

Objekt je entita, ktorá ma stav, správanie a indentitu.

$lada = new Car();

Trieda

Implementácia objektu, tvorí typ objektu.

class Car extends Thing implements Drive{}

Trieda (ďaľšie štúdium)

Vonkajsie "API" triedy a *conditions tvoria contract.

  • preconditions
  • postconditions
  • invariants

Interface

Spolu s triedou určuje typ objektu. Zaručuje isté správanie objektu.

Nepochopený génius!
interface Drive {}

Abstrakcia

Pri návrhu systému používame abstrakciu reálnych objektov.

Keď si napíšeme usecases, podst. mená sú dobrými kandidátmi na triedy.

Každá ponuka má niekoľko položiek. Položka sa dá vložiť do košika.

Polymorfizmus

Polymorfizmus je mechanizmus, ktorý umožňuje objektom nadobúdať rôzne typy alebo typu rôzne reagovať na volanie metódy. 

$programmer = new Programmer();
$human = new Programmer();
$programmer->dance();
$human->dance();

Úlohy

  1. Navrhnite štruktúru systému xml feedov.
  2. Navrhnite štruktúru cache knižnice.
  3. Navrhnite štruktúru fazetového vyhľadávania (filter).

Počítajte s tým, že data dostanete od vopred od IResource. Všetky návrhy musia byť ľahko rozšíriteľné (nový feed, cache driver, nový filter). Použite triedy, abstraktné triedy, interfaces. Metódy neimplementujte. Môžete použiť private a protected atribúty a metódy na naznačenie spôsobu implementácie.
